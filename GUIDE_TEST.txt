1. Ouvrir phpMyAdmin : http://localhost/phpmyadmin

2. Sélectionner la base de données "projet_valorant" dans le menu de gauche

3. Cliquer sur l'onglet "SQL" en haut

4. Copier-coller ce code SQL :

   ┌──────────────────────────────────────────────────┐
   │ USE projet_valorant;                             │
   │                                                   │
   │ ALTER TABLE games                                 │
   │ CHANGE COLUMN image_url image VARCHAR(255);      │

5. Cliquer sur "Exécuter"

6. Vous devriez voir : "1 ligne(s) affectée(s)"
